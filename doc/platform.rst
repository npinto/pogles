:mod:`pogles.platform`
======================
.. module:: pogles.platform
    :synopsis: Creates platform specific native windows.

The :mod:`pogles.platform` module contains platform specific bindings for
creating native windows.  Currently the Raspberry Pi and X11 platforms are
supported.  Usually the X11 platform will be used for development before
porting to a specific embedded platform.
